# Promise

Promise简单说就是一个容器，里面保存着未来才会结束的事件（通常是一个异步操作）的结果，Pormise是一个对象，从它可以获取异步操作的消息。Promise提供统一的api，各种异步操作都可以用同样的方法进行处理。 Promise的三种状态（pending(进行中),fulfilled(已成功),rejected(已失败)） ES6规定Promise是一个构造函数，用来实现Promise实例

const promise=newPromise(function(resolve,reject){
    if(/* 异步操作成功 */){
        resolve(value);
    }else{
        reject(error)
    }
})

Promise函数接收一个函数作为参数，该函数的两个参数分别是resolve，reject。他是两个函数，由javascript提供，不必自己部署。resolve、reject的作用分别是将Promise的状态从未完成变为成功/失败 Promise实例生成后，可以用then方法分别指定resolve，rejected状态的回调函数。

promise.then(function(value){

},function(error){

}
})

Promise创建后就会立即执行。

let promise = new Promise(function(resolve, reject) {
  console.log('Promise');
  resolve();
});

promise.then(function() {
  console.log('resolved.');
});

console.log('Hi!');
// Promise
// Hi!
// resolved

# Promise.prototype.then()

then方法接受两个函数作为参数，then方法的第一个参数是resolved状态的回调函数，第二个参数是rejected状态的回调函数。 then方法返回一个新的promise实例（不是原先的），因此可以采用链式写法。

# Promise.prototype.finally()

finally()方法不管最后状态如何，都会执行的方法。该方法是ES2018引入标准的。

promise
.then(res=>{})
.catch(res=>{})
.finally(res=>{})

# Promise.prototype.all()

Promise.all方法用于将多个Promise方法，包装成一个新的Promise实例

const p = Promise.all([p1, p2, p3]);

只有所有的实例的状态变为fulfilled，或者有一个变成reject，才会调用Promise.all()后面的回调函数

# Promise.prototype.race()

Promise.race()方法同样是多个Promise实例，包装成一个Promise实例。

const p = Promise.race([p1, p2, p3]);

只要有一个实例的状态发生变化，P的状态就跟着变化，那个率先变化的Promise的返回值，就是传给p的回调参数。

# Promise.allSettled()

Promise.allSettled()方法接受一组 Promise 实例作为参数，包装成一个新的 Promise 实例。只有等到所有这些参数实例都返回结果，不管是fulfilled还是rejected，包装实例才会结束。 该方法返回的新的 Promise 实例，一旦结束，状态总是fulfilled，不会变成rejected。状态变成fulfilled后，Promise 的监听函数接收到的参数是一个数组，每个成员对应一个传入Promise.allSettled()的 Promise 实例。 有时候，我们不关心异步操作的结果，只关心这些操作有没有结束。这时，Promise.allSettled()方法就很有用

# 实现一个pormise方法

首先实现构造函数的框架如下：

function Promise(executor) {
    var self = this;
    self.status = 'pending';
    self.data = undefined
    self.onResolvedCallback = []
    self.onRejectedCallback = []
    executor(resolve, reject) // 执行executor并传入相应的参数

    function resolve(value) {
        if (self.status === 'pending') {
            self.status = 'resolved';
            self.data = value;
            for (var i; i < self.onResolvedCallback.length; i++) {
                self.onResolvedCallback[i](value)
            }
        }
    }

    function reject(reason) {
        if (self.status === 'pending') {
            self.status = 'rejected';
            self.data = reason;
            for (var i; i < self.onResolvedCallback.length; i++) {
                self.onRejectedCallback[i](value)
            }
        }
    }
    try {
        executor(resolve, reject)
    } catch (error) {
        reject(error)
    }
}
