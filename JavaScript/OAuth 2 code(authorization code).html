# OAuth 2 code实现单点登录(authorization code)

# 1.OAuth2 单点的登录的实现逻辑（白话版）

  1. OAuth2首先会将浏览器跳转到一个登录后的URL页面（即redirect_url），并在URL后添加code
  2. 页面跳转后，前端根据这个code请求自己的后端服务器，后端根据code判断用户是否存在。如果存在，将对应的登录信息返回到前端（跟账号密码登录一样）。如果不存在，返回错误信息。
  3. 前端将登录的信息储存，跳转到登陆后页面。

# 2.具体代码(vue router guard)

if (!(to.name === 'login' || to.name === 'register' || to.name === 'home')) {
    // auth2进入逻辑
    if (store.state.authModule.sysAuthInfo.value.authStatus === 'ENABLE') {
      if (to.query?.code) {
          // 如果链接地址后有code参数
        const request = {
          ...apis.AUTH_CODE_LOGIN,
          params: {
            code: to.query?.code
          }
        };
        // code登录逻辑开始
        await new Promise((resolve, reject) => {
          Vue.prototype
            .$request(request)
            .asyncThen((resp) => {
              if (resp.code === 'ok') {
                store.commit('...');
                resolve();
              }
            })
            .asyncErrorCatch(() => {
              next('/login');
            });
        });
      } else if (!store.state.authModule.token) {
        // 如果无token跳转到单点登录页面
        next(false);
        const { authObj } = '...'
        const url = `${authObj.authorizationUrl}?response_type=code&client_id=${authObj.clientId}&redirect_uri=${authObj.redirectUrl}`;
        await window.location.replace(url);
        // 此处注意 window.location.replace实际上是一个异步函数，网速不够快的时候会造成闪登录页等奇怪问题
        return;
      }
    }
